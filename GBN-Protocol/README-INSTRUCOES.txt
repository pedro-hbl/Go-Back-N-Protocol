Intruções para utilização do software:

1- Abra 2 terminais.

2- Acesse a pasta do projeto ( cd Go-Back-N-master )

3- Compile no primeiro terminal:  javac Receiver.java

4- Compile no segundo terminal: javac Sender.java

5- Execute no primeiro terminal: java Receiver

6- Execute no segundo terminal: java Sender

7- Utilize o software pelo segundo terminal.
